lists

there are three types of lists...

  1. ordered List

    1. is created using the < ol > tag
    2. displays numbered bullet points

  2. Unordered List

    • is created using the < ul > tag
    • displays non numbered bullet points

  3. Defined List

    4. defined lists show terms and thier definitions < dt > defines the define term of the list entry
    dd defines the definition of the term
    the < li > function is not used in these types of lists
