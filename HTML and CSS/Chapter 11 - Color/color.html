Color

Foreground Color

  • The "color" property allows you to specify the color of text inside an element. You can specify any color in CSS in one of three ways

    1. RGB Values: color: rgb(100,100,90);

    2. Hex Code: color: #ee3e80;

    3. Color Names: color: DarkCyan

    <>

Background Color

  • CSS Treats each HTML element as if it appears in a box, and the "background-color" property sets the color of the background for that box.

  • You can specify the background color in the sameway you specify forground color; rgb, hexcode and color names.

  • If you do not specify a background, the background is transparent.

  • To set the background color of an element, you add background-color: plus rgb, hex, or color name.

CSS3: Opacity

  • CSS3 introduces the "opacity" property which allows you to specify the opacity of an element and any of its child elements. The value is a number between 0.0 and 1.0.

  • The CSS3 "rgba" property allows you to specify a color, just like you would an rgb value but adds a fourth value to indicate opacity. This value is known as the apha value. The rgba value will only affect the element on which it is applied (not child elementss)

CSS3 hsl and hsla

  • The "hsl" color property has been introduced in CSS3 as an alternative way to specify colors. The value of the property starts with the letters "hsl" with individual values within the curly brackets representing hue (between 0 and 360), saturation(percentage) and lightness(percentage).

  • The "hsla" color property allows you to specify color properties using hue saturation and lightness the same as using hsl, but also adds a fourth value that represents transparentcy (just like the rgba property) which stands for alpha.
