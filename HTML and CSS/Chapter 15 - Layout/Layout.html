Layout

Normal Flow - position: static;

  • In normal flow, each block-level element sits on top of the next one. Since this is the default way in which browsers treat HTML elements, you do not need a CSSproperty to indicate the elements should appear in normal flow, but the syntax would be: position: static;

  • I have not specified a width: property for the heading element, so you can see how it streatches the width of the entire browser window by default.

  • The paragraphs have been restricted to 450 pixels wide. This shows how the elements in normal flow start on a new line even if they do not take up the full width of the browser window.

Relative Positioning - position: relative;

  • Relative positioning moves an element in relation to where it would have been in normal flow.

  • For example, ypu can move it 10 pixels lower than it would have been in normal flow or 20% to the right.

  • You can indicate that an element should be relativly positioned by using position: relative;

  • You can then use the offset properties (top and bottom and left or right) to indicate how far to move the element from where it would have been in normal flow.

    • To move the box up or down, you can use either the top or bottom properties.

    • To move the box horizonatally, you can use either the left or right properties.

Absolute Positioning - position: absolute;

  • When the position property is given a value of absolute, the box is taken out of normal flow and no longer affects the position of other elements on the page. (They act like its not there.)

  • The box offset properties (top, bottom, left, right) specify where the element should appear in relation to containing element.

  • In this example, the heading has been positioned 500 pixels from its left edge. The width of the heading is set to be 250 pixels wide.

  • The width property has also been applied to the < p > elements in this example to provent the text from overlapping and becoming unreadable.

Fixed Positioning - position: fixed;

  • Fixed positioning is a type of absolute positioning the requires the position property to have a value of fixed.

  • It positions the element in relation to the browser. Therefore, when a user scrolls down a page, it stays in the exact same place.

  • To control where the fixed position box appears in relation to the browser window, the box offset properties arw used (top, bottom, left, right)

  • The < p > elements are in normal flow and ignore the space the the heading element would have taken up. Therefore, the margin: top; has been used to push the first < p > element below where the fixed heading is sitting.
