HTML5 Video and Audio

Adding video your pages

  • The < video > element has a number of attributes that allow you to control video playback:

    • "src": This attribute specifies the path to the video

    • "poster": This attribute allows you to specify and image to show while the video is downloading or until the user tells the video to play.

    • "width" and "height": These attributes specify the width and the height of the video.

    • "controls"; This attribute indicates whther the browser should supply its own controls for playback.

    • "autoplay": This specifies whether the video should start playing when the page loads.

    • "loop": This states whether the video file should loop or not

    • "preload": This tells the browser what to do when the page loads. it can have one of three values:

      • "none": The browser should not load the video until the user presses play

      • "auto": The browser should download the video when the page loads

      • "metadata": The browser should just collect information such as the size, first frame, tracklist and duration

  • For example: < video src = "myvideo.mp4" poster = "images/teddy.jpg" width = "400" height = "300" preload = "auto" controls loop > < p >A vido that I have used on my website < / p>< / video >

Multiple video sources

  • To specify the location of the file to be played, you can use the < source > element inside the < video > element. (This should replace thesrc attribute on the opening < video > tag.) e.g. < video width = "400" height = 300 preload = "auto" controls >< source src = "video/dontask.mp4" / >< p >this is a video< / p >< / video >

  • You can use multiple < source > elements to specify that the video is available in different formats.

  • You should use the "type" attribute to tell the borwser what format the video is in e.g. < video src = "myvideo.mp4" type ='video/mp4; codecs = "vp8, vorbis"' >< / video >

  • The "codec" thatwas used to encode the vidoe is supplied within the type attribute. Note the use of single quotes, as well asdouble qutes in the type attribute, when it is supplied.

  • If the browser does not support the format of video used, it will display any text the is between the two < video > tags.
