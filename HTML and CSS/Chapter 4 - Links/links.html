links

  • links are created using the < a > element. users can click on anything between to opening and closing tag.

  • you specify which page you want to link to by using the href attribute in the opening tag. e.g. < a href = "http://www.bbc.co.uk" > bbc website < / a > you need the full site name, http and all

  • you use the exact same method to link to to other pages within your website. instead of using a website in the href, you use your html document. it just needs to be in the same folder as the current page. e.g. < a href = "contact.html" > contact us < / a >

  1. here is a link to the bbc website!
  2. here is a link to another page!

relative urls...

  • same folder = < a href = "pagename.html">

  • child folder (in a folder) = < a href = "folder/pagename.html">

  • grandchild folder (folder in a folder) = < a href = "folder/folder/pagename.html">

  • parent folder (in previous folder) = < a href = "../pagename.html">

  • grandparent folder (in folder before previous folder) = < a href = "../../pagename.html">

email links...

  • email links are used by putting mailto: examplemail@email.com just after href in the opening < a > tag. e.g. < a href = "mailto: example@example.com"> email person < / a>

  • email me! click here!

links in new tab...

  • to open a page in a new tab, add target = "_blank" after the href stuff. e.g. < a href = "http://www.bbc.co.uk" target = "_blank" > opens in new tab < / a>

  • here is a link to the bbc website! (opens in new tab)

links to specific part of page...

  • to link to a specific part of another page, an id must be assigned to the desired section before you can jump to it e.g. < h1 id = "top" > top < h1 >

  • you can then link to that part of the page using a href with a # in front of the id name. e.g. < a href = "#top">top< / a >

  • you can also link to another websites id tags, as long as the have them such as: < a href = "http://www.bbc.co.uk/#top">top of bbc page< / a >
