Specifying Typefaces

  • The "font-family" property allows you to specify the typeface that should be used for any text inside the element(s) to which a CSS rule applies

  • You can specify a list of fonts seperated by commas (arial, verdana ext.) so that if a user does not have your first choce of typeface installed, the browser can try and an alternative font from the list given

Size of Type

  • The "font-size" property allows you to specify a size for the font. There are several ways to specify the size of a font, the most comon are:

    1. Pixels - font-size: 12px;

    2. Percentages: font-size = 200%; (Based on a default browser size of 16 pixels)

    3. EMS (Width of the letter 'M') - font-size: 1.5em;

More Font Choice

  • "@font-face" allows you to use a font font even if it is not installed on the computerof the person browsing, by allowing you to specofy a path to a copy of whe font which will be downloaded if it is not on the users machine.

  • You add the font to a style sheet using the @font-face rule with the properties of "font-family" (the name of the font) and src (the source of the font file). e.g. @font-face {font-family: 'ChunkFiveRegular'; src: url('fonts/chunkfive.eot'); }

  • Different browsers support different font types, this means that you must import the correct font file type for it to work in the desired browser, the font file types for each browser are:

    • .eot - IE 5-8, IE 9+

    • .woff - Chrome 6+, Firefox 3.6+, IE 9+

    • .ttf/.otf - Chrome 6+, Firefox 3.5, Firefox 3.6+, IE 9+, Opera 10+, Safari 3.1+, iOS 4.2+

    • .svg - Chrome (All), Opera 10+, Safari 3.1+, iOS < 4.2, iOs 4.2+

  • When importing, you will have to add a "format" attribute to the @font-face code stating the type of file that you are including. for example: @font-face {font-family: 'ChunkFiveRegular'; src: url('fonts/chunkfive.eot'); format('embedded-opentype') }

  • In order to have the font load on all browsers, you will need to import all file types to make it compatible. You do this by just adding more "src" and "format" attributes to the @font-face tag.

  • Also, the font files must be next to the CSS file that they are being used in

Bold

  • The "font-weight" property allows you to create bold text. There are two values that thisproperty commonly takes:

    1. font-weight: normal; - This causes the text to appear at a normal weight.

    2. font-weight: bold; - This causes text to appear in bold.

Italic

  • If you want to create itallic text, you can use the "font-style" property. There are three values this property can take

    1. font-style: normal; - This causes the text to appear in a normal style.

    2. font-style: italic; - This causes the text to appear in a italic style.

    3. font-style: oblique; - This causes the text to appear in an oblique sty;e.

Uppercase and Lowercase

  • The "text-transform" property is used to change the case of text giving it one of the following values:

    1. text-transform: uppercase; - Causes text to appear in upper case.

    2. text-transform: lowercase; - Causes the text to appear in lower case.

    3. text-transform: capitalize; - Causes the first letter of each word to appear capitalized

Underline and Strike

  • The "text-decoration" property allows you to specify they following values:

    • text-decoration: none; - Removes all text decoration.

    • text-decoration: underline; - underlines the specified text.

    • text-decoration: overline; - overlines the specified text

    • text-decoration: line-through; - puts a line through the specified text.

Line Height

  • Leading (Ledding) is used to define the vertical spece in between lines of text. to set this, we use: line-height: 1.4em;

  • Becuase the user can adjust the default size of text in thier browser, the value of the line-height is best given in ems. A good starter setting is between 1.4 and 1.5 ems

Letter and Word spacing

  • letter-spacing: 1.3em; is used to set the space in ems between letters.

  • word-spacing: 1em; is used to set the space between words. The default it often around 0.25em

Text Alignment

  • The "text-align" property allows you to control the algnments of text. The property can take one of four values:

    • text-align: left; - The text should be left aligned

    • text-align: right; - The text should bed right alighned

    • text-align: center; - The text should be in the center

    • text-align: justify; - The text should be in justified format

Vertical Align

  • The "vertical-align" propertyis commonly used with inline elements such as images, "strong" and "em". When used with these elements it acts similar to the html "align" attribute. The values it can take are:

    • baseline

    • sub

    • super

    • top

    • text-top

    • middle

    • bottom

    • text-bottom

  • It can also take a length (usually specifed in pixels or ems) or a percentage of line height

Indenting Text

  • The "text-indent" property allows you to indent the first line of text within an element. The amound you want the line to be intented by can can be specified in a number of ways but it is usually given in pixels (px) or ems (em). Positive (right) and negative (left) values can be given. e.g. text-indent: 200px;

CSS3: Drop Shadow

  • The "text-shadow" property is used to create a drop shadow, which a dark version of the word just behind it abs slightly offset. It can also be used to create an emobossed effect by adding a shadow that is slightly lighter then the text.

  • The value of this property can take three lengths and a color for the drop shadow.

    1. The first length indicates how far to the left or right the shadow should fall - text-shadow: 1px 1px 0px #000000;

    2. The second value indicates the distance to the top or bottom that the shadow should falltext-shadow: 1px 1px 0px #000000;

    3. The third value is optional and specfies the amount of blur that should be applied to the drop shadow. text-shadow: 1px 1px 0px #000000;

    4. The fourth value is the color of the drop shadow. text-shadow: 1px 1px 0px #000000;

First letter or line

Styling Links

  • Browsers tend to show links in blue with an underline by default. In CSS there are two psuedo-classes that allow you to set different styles for links have and have not yet been visited.

    1. a:link - sets styles for link that have not been visited.

    2. a:visited - sets styles for links that have been visisted.
