Box Dimensions

  • By default, a box is sized just big enough to hold it contents. To set your own dimensions for a box, you can use the "height:" and "width:" properties

  • The most popular way to specify box size is in pixels, percentages or ems. When using percentages, the box size is relative to the size of the browser window. Or if it already inside a box, it is relative to the size of that box. With ems, the box is based on the size of the text within it

Limiting Height

  • In the same way that you might want to limit the width of an box on a page, you may also want to limit the height of it. This is acheived using the "min-height" and "max-height" properties.

Overflowing Content

  • The "overflow:" property tells the browser what to do if the content contained within a box is larger than the box itself. It can have one of two values:

    1. overflow: hidden; - This property simply hides any extra content that does not fit in the box

    2. overflow: scroll; - This property adds a scrollbar so the users can scroll to see the missing content.

Border Width

  • The "border-width:" property is used to control the width of a border. The value of this property can either be given in pixels or on of three values: "thick", "thin", or "medium". You cannot use percantages with this property

  • You can control the individual size of borders using four seperate properties:

    1. border-top-width:

    2. border-right-width

    3. border-left-width

    4. border-bottom-width

  • You can also specify different widths for the four border values in one property like so: "border-width: 4px 6px 4px 3px;"

  • The values appear in clockwise order; top, right, bottom, left

Border Style

  • You can control the style of a border using the "border-style" property. This property can take the following values:

    1. border-style: solid; - A single solid line.

    2. border-style: dotted; - A series of squared dots (If your border is 2px wide, then the dots are 2px squared with a 2px gap between each dot)

    3. border-style: dashed; - A series of short lines.

    4. border-style: double; - Two solid lines (the value of the "border-width" property creates the sum of the two lines)

    5. border-style: groove; - Appears to be carved into the page.

    6. border-style: ridge; - Appears to stick out from the page.

    7. border-style: inset; - Appears embedded into the page.

    8. border-style: outset; - looks like it is coming out of the screen

    9. border-style: none; - No border is shown

  • You can individually change the styles of different borders using, "border-top-style", "border-right-style", "border-bottom-style", and "border-left-style"

Border Color

  • You can specify the color of a border using either RGB colors, Hexcodes or CSS color names. e.g. border-color: red;

  • It is possible to individually control the colors of the borders on different sides of the borders using "border-top-color", "border-right-color", "border-bottom-color", and "border-left-color"

  • It is alos possible to use a shorthand to control all four border colors in one propery. e.g.: "border-color: blue red green orange;" with the values appearing in a clockwise order.

Padding

  • The "padding:" property allows you to specify how much space should appear between the content of an element and its border.

  • The value of this property is mostly specified in pixels, however if it is done in percentage, it is a percentage of its containing box. e.g. padding: 12px;

  • You can specify different padding for each side of th box using: "padding-top:", "padding-right:", "padding-left:", "padding-bottom"

  • Or, you can use shorthand to specify padding for each of the sides clockwise. e.g. padding: 12px, 3px, 5px, 12px;

Margin

  • The "margin:" property control the gap between boxes. It's value is commoly given in pixels, although you may use percentages or ems.

  • If the width of a box is psecified then the margin is added to the width of the box.

  • You can specify values for each side of the box using; "margin-top:", "margin-right:", "margin-bottom:" and "margin-left:"

  • You can also use shorthand to set a different margin for each side (in a clockwise order) e.g. "margin: 1px 4px 6px 7px;"

Centering Content

  • If you want to center a box on the page (or inside th element it sits in,) You can set the "left-margin" and "right-margin" to "auto"

  • In order to denter a box, you need to set a "width" for the box otherwise it will take up the full size of the page

  • Once you have set the width of the box, setting the left and right margin to "auto" will make the browser put an equal gap on each side of the box, centering it

Change Inline/Block

  • The "display" property allows you to turn an inline element into a block-level element or vice versa, and can also be used to hide an element from the page

  • The value these properties can take are:

    1. display: inline; - This causes a black-level element to act like an inline element

    2. display: block; - This causes an inline element to act like a black level element

    3. display: inline-block; - This causes a block-level element to flow like an inline element, while retaining other features of a block like element.

    4. display: none; - This hides an element from a page.

Hiding Boxes

  • The "visibility" property allows you to hid boxes from users but it leaves a space where the element would have been.

  • This property can take two values

    1. visibility: hidden; - This hides the element.

    2. visibility: visible; - This shows the element

  • If you do not want a blank space to appear, then you should use the "display" property with a value of "none" (display: none;) instead.
