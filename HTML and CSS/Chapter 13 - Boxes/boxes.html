Boxes

Box Dimensions

  • By default, a box is sized just big enough to hold it contents. To set your own dimensions for a box, you can use the "height:" and "width:" properties

  • The most popular way to specify box size is in pixels, percentages or ems. When using percentages, the box size is relative to the size of the browser window. Or if it already inside a box, it is relative to the size of that box. With ems, the box is based on the size of the text within it

Limiting Height

  • In the same way that you might want to limit the width of an box on a page, you may also want to limit the height of it. This is acheived using the "min-height" and "max-height" properties.

Overflowing Content

  • The "overflow:" property tells the browser what to do if the content contained within a box is larger than the box itself. It can have one of two values:

    1. overflow: hidden; - This property simply hides any extra content that does not fit in the box

    2. overflow: scroll; - This property adds a scrollbar so the users can scroll to see the missing content.
