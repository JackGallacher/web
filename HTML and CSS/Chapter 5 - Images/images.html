Images

  • to add an image to a page, the < img > tag is used.

  • inside the image tag the source of the file is inputted, for example < img src = "myimg.jpg" / >the img tag does not have a closing tag and only uses a forwardslash to close it

  • the alt attribute provides a text description of the image if it cannot be displayed e.g. < img src = "myimg.jpg" alt = "a picture of my image"/ >

  • the title attribute is used as a tooltop when the image is hovered over e.g. < img src = "myimg.jpg" alt = "a picture of my image" title = "my image!" / >

  • the height and width of the image is specified using the height and width attributes e.g. < img src = "myimg.jpg" alt = "a picture of my image" title = "my image!" height = "600" width = "450" / >
