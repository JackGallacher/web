Extra Markup

Doctypes

  • Becuase there have been several versions of html, each webpage should begin with a "DOCTYPE" declaration to a tell a broser which type of html the page is using e.g. < !DOCTYPE html >

Comments

  • If you want to add comments to your code that will not be visible in the users browsers, you can add the text between these characters: < !-- comments go here -- >

ID Attribute

  • Every HTML element can carry the "id" attribute. It is used to uniquely identify that element on the page. e.g. < h1 id = "page title">Title< / h1 >

Class Attribute

  • Every HTML element can also have a "class" attribute. Sometimes, rather than uniquely identifying one element within a document, you will want a way to identify several elements on the page. e.g. < p class = "important info" > important paragraph stuff < / p>

Block Elements

  • Some elements will always appear to start on a new line in the browser window. These are known as "block level" elements.

  • Examples of block elements are < h1 >, < p >, < ul > and < li >

Inline Elements

  • Some elements will always appear to continue on the same line as thier neighbouring elements. These are known as "inline" elements.

  • Examples of inline elements are < a >, < b >, < em > and < img >
