Lists, Tables and Forms

Bullet Point Styles

  • The "list-style-type" property allows you to control the shape or syle of a bullet point. (also known as a marker)
  • It can be used on rules that apply to the < ol >, < ul > and < li > elements.
  • For unordered lists, you can use the following values:
    • list-style-type: disk;
    • list-style-type: circle;
    • list-style-type: square
    • list-style-type: none;
  • For ordered lists, you can use the following values:
    1. list-style-type: decimal
    2. list-style-type: decimal-leading-zero;
    3. list-style-type lower-alpha;
    4. list-style-type: upper-alpha;
    5. list-style-type: lower-roman;
    6. list-style-type: upper-roman;

Images for Bullets

  • You can specify an image to act as a bullet point using the "list-style-image:" property
  • The value starts with the letters "url" and is follewed by a pair of parentheses. Inside the parentheses, the path to the image is given inside double quotes. e.g. list-style-image: url("path of image");
  • This property can be used on rules that apply to the < ul > and < li > elements.

Positioning the Marker

  • Lists are intended into the page by default and the "list-style-position" property indicates whether the market should appearf on the inside or the outside of the box containing the main points. This property can take one of two values:
    • list-style-position: outside; - The marker sits to the left of the block of text (This is defult behaviour when this property is not used).
    • list-style-position: inside; - The marker sits inside of the box text (which is indented)

List Shorthand

  • As with serveral of the other CSS properties, there is a property that acts as a shorthand for the list styles. "list-style" allows you to express the markers style, image and position in any order.
  • list-style: square inside;

Border on Empty Cells

  • If you hve empty cells in your table, you can use the "empy-cells"property to specify whether or not thier borders should be shown. It can take one of three values:
    • empty-cells: show; - This shows the border of any empy cells.
    • empty-cells: hide; - This hides the borders of any empty cells.
    • empty-cells: inherit; - If you have one trblae nested inside another, inherit the rules of the containing element.

Cursor Styles

  • The "cursor" property allows you to control the type of mouse cursor tht should be displayed to users.
  • For example, on a form you might set the cursor to be a hand when the user hovers over it. here are the most commonly used values for this property:
    • cursor: auto;
    • cursor: crosshair;
    • cursor: default;
    • cursor: pointer;
    • cursor: move;
    • cursor: text;
    • cursor: wait;
    • cursor: help;
    • cursor: url("cursor.gif");

Gaps Between Cells

  • The "border-spacing" property allows you to control the distance between ajacent cells. By default, browsers often leave a small gap between each table cell. "border-spacing" allows you to control this gap
  • The value of this property is usually specified in pixels. You can specify two values if desired to specify seperate numbers for horizontal and vertical spacing.
  • You can use the "border-collape" property to prevent inside lines of tables cells being double that of outside lines due to sitting next to each other. The "border-collapse" property has two possible values:
    1. border-collapse: collapse; - Borders are collapsed into a single border where possible
    2. border-collape: seperate; - Borders are detched from each other where possible.
