Tables...

  • a basic table is created within the < table >tag.

  • a row in that table is created using the < tr > tag in the same manner an ordered or unordered list is made.

  • the data in each row is inputted using the < td > tag, just like a list element is added.

  • table headings are placed within the < th > tag, within this the scope attribute is used where you can specificy if it is a coloumn header (col) or row (row). for example; < th scope = "col" >this is my column< / th >

  • to add a blank row or column, just leave the space blank between then opening and closing tags, it helps to format the table

  • to make a cell span multiple columns, use the colspan attribute in the tabe data tag. for example: < td colspan = "2">55< / td >

  • to make a cell span multiple rows, use the rowspan attribute in the tabe data tag. for example: < td rowspan = "2">55< / td >

  • for long tables or to add a bit more order, you can use the < thead >, < tbody >, and < tfoot >, headings and such would go in the head, data in the body and footers in the foot.
